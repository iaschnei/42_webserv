# 42_webserv
A simple webserver in C++, hosting a custom website, made in collaboration with [Acsts](https://github.com/Acsts)

Usage: 
```sh
make && ./webserv
```
Then simply visit ```https://localhost:4243``` on your browser ! Enjoy the dog pictures
